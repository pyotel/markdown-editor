# 📝 Markdown Editor

간단하고 빠른 마크다운 에디터

## ✨ 특징

- ✅ **양쪽 편집 모드** - Markdown / WYSIWYG 전환
- ✅ **Side-by-side 미리보기** - 왼쪽 편집, 오른쪽 실시간 렌더링
- ✅ **파일 저장/열기** - 로컬 파일 시스템
- ✅ **마크다운 도구** - 버튼으로 문법 입력
- ✅ **100% 오프라인** - 인터넷 불필요
- ✅ **키보드 단축키** - Ctrl+S 저장, Ctrl+O 열기

## 🚀 실행 방법

### 개발 모드
```bash
npm install
npm start
```

### Windows EXE 빌드
```bash
npm install
npm run build
# → dist/Markdown Editor-1.0.0.exe
```

### Linux AppImage
```bash
npm run build-linux
# → dist/Markdown Editor-1.0.0.AppImage
```

## 🛠️ 기술 스택

- **Electron** - 데스크톱 앱
- **EasyMDE** - 마크다운 에디터 (SimpleMDE 후속)
- **순수 HTML/CSS/JS** - 빌드 불필요

## 📝 사용법

1. **편집** - 왼쪽에 마크다운 입력
2. **미리보기** - 👁️ 버튼으로 side-by-side 활성화
3. **저장** - 💾 버튼 또는 Ctrl+S
4. **열기** - 📂 버튼 또는 Ctrl+O

## 🎨 마크다운 도구

- 굵게, 이탤릭, 제목
- 인용구, 리스트
- 링크, 이미지
- 코드 블록
- 전체화면 모드
